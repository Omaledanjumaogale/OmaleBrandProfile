In relational database design, which key constraint ensures that a combination of attributes uniquely identifies every row in a table?
AForeign Key
BComposite Primary Key
CCandidate Key
DIndex Key
✅ B is correct. A Composite Primary Key uses two or more columns together to uniquely identify each row. ❌ A is a reference key to another table. ❌ C is a candidate — not yet designated primary. ❌ D is a performance optimiser, not a uniqueness enforcer.
